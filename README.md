# Digital Pixel Clock

## Creating secret.h
Within your DigitalPixelClock directory, create a `secret.h` header file.  

Populate as follows:
```c++
// Network Config
#define WIFI_SSID "" // Your WiFi SSID (2.4GHz)
#define WIFI_PASS "" // Your WiFi Password
```
